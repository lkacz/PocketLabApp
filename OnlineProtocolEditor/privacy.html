1. Introduction

Pocket Lab App (“the App”) is an open-source Android application for smartphone-guided experiments. The App works fully offline and does not collect, transmit, or share any personal data. All information stays on the device controlled by the researcher or user.

2. Data collection and use

The App does not:

  • collect personal or sensitive user data,
  • send data to external servers,
  • use analytics, advertising, or tracking SDKs,
  • require user accounts or registration.

All experiment protocols, responses, logs, and output files are stored locally on the device.

3. Permissions

The App may request device permissions (e.g. storage access, alarms/notifications) solely to read protocol files and save locally collected data. No information accessed with these permissions is transmitted off the device.

4. Security

Because the App does not connect to the internet, the main security measure is physical protection of the device and responsible handling of exported data by the researcher.

5. Open-source

The App is distributed under the GNU GPLv3 license. Source code and documentation are available at: https://github.com/lkacz/PocketLabApp.

6. Changes

Any future updates to this policy will be published under the same repository and/or with the App.
