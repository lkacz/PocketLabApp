Immersive Briefing

This custom HTML slide accompanies the full PoLA command tutorial. Use it to review the segments that were just demonstrated and to understand which resource files are referenced by the multimedia placeholders (<audio>, <video>, and <html> tokens) within the protocol.

Highlighted features

  • Appearance configuration (colors, sizes, alignments, transitions)
  • Randomization blocks and randomized input fields
  • Timers with audio alerts and hold-protected continue buttons
  • Branching with labels and goto commands
  • Inline multimedia references for narration, ambience, and video
Place the referenced media files in the selected Resources folder before running the protocol:
  • tutorial_voiceover.mp3 — narration for the welcome slide
  • tutorial_ambient_loop.wav — subtle ambience looped in the background
  • tutorial_checkpoint_video.mp4 — short clip illustrating video playback
  • tutorial_success_chime.wav — chime played by the TIMER_SOUND command

Adjust these filenames in the protocol if you prefer different assets.

When you close this HTML view, return to the PoLA tutorial to continue with the remaining command demonstrations.
